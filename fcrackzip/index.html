fcrackzip

fcrackzip adalah cracker kata sandi cepat yang sebagian ditulis dalam assembler. ia mampu memecahkan file zip yang dilindungi kata sandi dengan serangan berbasis brute force atau kamus, secara opsional menguji dengan unzip hasilnya. itu juga dapat memecahkan gambar cpmask.

install fcrackzip

perintah :

apt-get update && apt-get install fcrackzip -y

penggunaan fcrackzip

perintah :

fcrackzip -v -u -D -p /path/wordlists /path/file zip
